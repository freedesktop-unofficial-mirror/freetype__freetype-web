The FreeType Auto-Hinting Resources Pages

Welcome to this new section of the FreeType site. We have created this page in order to gather opinions, information, experience, comments and ideas regarding the design and implementation of a new module called the FreeType Auto-Hinter.

The auto-hinter (that we'll simply call "the hinter" in the rest of this pages) is a module used to align glyph outlines to the pixel grid in order to tremendously improve glyph images when they are finally scan-converted, either to a monochrome bitmap, or an anti-aliased one.

We have set up a public mailing list to discuss this topic. To subscribe to the list and be able to participate, please visit the page :

http://lists.sourceforge.net/mailman/listinfo/freetype-autohint
