FreeType 1 Download

We do not distribute binary packages for FreeType 1. You can however access its source code using one of these methods:

  • Download it directly
    By using our SourceForge page, or FTP mirrors. Details are available on this page

  • Using our anonymous CVS server:
    Simply follow these instructions, using the module name "freetype" to upload the current snapshot of the FreeType 1 sources to your machine.

  • Using our ViewCVS web page
    Located at the following address:

    http://cvs.savannah.gnu.org/viewvc/?root=freetype
